Bag-of-Visual-Words Feature Extraction using Graph Convolutional Networks

Published on: July 12, 2024

This paper presents a novel deep learning framework tailored for remote sensing image analysis, overcoming the inherent limitations of traditional Bag-of-Visual-Words approaches. By integrating a Convolutional Neural Network (CNN) with a Graph Convolutional Network (GCN), the method first leverages the CNN to extract high-level spatial features and capture prominent visual patterns in remote sensing images. These features are then processed through the GCN, which learns complex spectral-spatial relationships by operating over a graph constructed from the image data. This hybrid architecture not only refines feature extraction by combining spatial layouts with intricate spectral details but also enhances performance in tasks such as image retrieval and classification.
